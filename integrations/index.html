Build Your Own Integration

Our REST API and webhooks make it simple to connect Clayflow with anything. Build custom integrations, sync data bidirectionally, and trigger workflows from any system.

Fully documented REST API with SDKs for JS, Python, Go, and Ruby
Real-time webhooks for events, changes, and triggers
OAuth 2.0 authentication with granular scopes
Read the Docs API Reference
PUT /api/v1/integrations 
{
  "name": "my-custom-app",
  "type": "webhook",
  "events": ["task.created", "task.updated"],
  "config": {
    "url": "https://api.myapp.com/webhooks",
    "auth": { "type": "oauth2" }
  }
}

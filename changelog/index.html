v2.5.0 — Real-time Collaboration

Released June 16, 2026

Latest

New Features

  • ● Real-time multi-user editing on dashboards with presence indicators.
  • ● New AI-powered report generator — describe what you need in natural language.
  • ● Custom role-based access controls for Enterprise plans.

Improvements

  • ● Dashboard load times improved by 40% with lazy-loading widgets.
  • ● Mobile responsiveness across all core views.
  • ● Search now supports fuzzy matching and advanced filters.

Fixes

  • ● Fixed export-to-PDF truncation on multi-page reports.
  • ● Resolved webhook timeout issue for payloads exceeding 1MB.
  • ● Corrected timezone display in scheduled report emails.

v2.4.0 — Deeper Integrations

Released May 28, 2026

New Features

  • ● Native Slack integration — receive alerts, reports, and summaries in channels.
  • ● Salesforce connector — sync accounts, opportunities, and contacts bidirectionally.
  • ● API endpoint for bulk data export (/v1/exports).

Improvements

  • ● Webhook retry logic with exponential backoff and dead-letter queue.
  • ● OAuth token refresh now handled automatically in integrations.

Fixes

  • ● Fixed CSV import failing on rows with special characters.
  • ● Resolved duplicate notification issue on recurring reports.

v2.3.0 — Analytics Overhaul

Released April 14, 2026

New Features

  • ● Completely redesigned analytics engine with custom metric builder.
  • ● Drill-down charts — click any data point to explore underlying records.
  • ● Scheduled report delivery via email, Slack, and webhook.

Improvements

  • ● Chart rendering performance improved by 60% on large datasets.
  • ● New chart types: funnel, heatmap, and waterfall.

Fixes

  • ● Fixed date range picker resetting on filter change.
  • ● Resolved currency formatting issues in multi-currency workspaces.

New Features
